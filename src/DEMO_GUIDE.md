# 🎮 NeuroPlay Demo Guide

**Quick 10-minute walkthrough for demonstrating the autism-friendly challenge quest game**

---

## 🚀 Quick Setup

- Clear browser localStorage if needed: `localStorage.clear()` in console
- Have app running and accessible
- Set volume for text-to-speech

---

## 📋 Demo Flow (10 minutes)

### 1️⃣ INTRO (1 min)

**Say**: "NeuroPlay is an autism-friendly challenge quest game with 15 therapeutic challenges, comprehensive accessibility features, and family management tools."

**Show**: Login screen with clean, colorful design

---

### 2️⃣ KID EXPERIENCE (4 min)

**Create Account**:
- Click "Sign Up"
- Username: `alex` | Password: `password123` | Age: `8`
- Click "Create My Account!"

**Dashboard Tour**:
- ✨ Point out personalized greeting, XP, and level
- 🎨 Click "Customize Avatar" - change hair, skin, outfit, accessories
- 😊 Click a mood emoji in Mood Check-In widget
- 🏆 Show Badge Collection (scroll through earned/locked badges)

**Play Challenge**:
- Click "Start Challenge" on Daily Challenge (or pick "Memory Match")
- Play a few rounds - show positive feedback
- Click "Need a Break?" button - show calming break screen
- Return and finish (or go back to dashboard)

**Accessibility Menu**:
- Click accessibility icon (♿)
- Toggle ON Text-to-Speech → click "Test Voice"
- Move Font Size slider to "Large"
- Toggle High Contrast Mode
- Switch theme to "Ocean Blue"
- **Say**: "6 accessibility options for diverse needs"

## 💬 One-Liner Answers

**Q: Who is this for?**  
A: Children with autism, ADHD, sensory processing disorders, ages 5-12

**Q: What makes it autism-friendly?**  
A: Text-to-speech, customizable themes, break reminders, reduced motion, high contrast, adjustable font sizes

**Q: Can parents track multiple kids?**  
A: Yes, one caregiver account manages unlimited children with quick-switching

**Q: How is data stored?**  
A: Currently localStorage (demo), production uses Supabase backend
